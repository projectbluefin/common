# GNOME 47 “Denver” Release Notes

**Release Date:** September 18, 2024

GNOME 47 introduces a range of new features, improvements, and polish across the desktop environment and its core applications. This release is dedicated to the organizers of GUADEC 2024.

![GNOME 45 Default Desktop](https://upload.wikimedia.org/wikipedia/commons/thumb/e/e0/Gnome-45-default-desktop.png/1024px-Gnome-45-default-desktop.png)

---

## Highlights

- **Accent Colors:**
  Customizable accent colors are now available! You can choose from several accent colors in the Appearance settings, letting you personalize the system beyond the classic blue.

![GNOME 46 Shell Overview](https://upload.wikimedia.org/wikipedia/commons/thumb/b/b5/Gnome-46-shell-overview.png/1024px-Gnome-46-shell-overview.png)

- **Improved Small Screen Support:**
  GNOME now automatically scales icons and UI elements for better usability on lower-resolution displays and small screens.

- **Screen Capture Hardware Encoding:**
  Record screens more efficiently with hardware encoding support for Intel and AMD GPUs, resulting in smoother performance and lower system load during screen recordings.

- **Persistent Remote Desktop Sessions:**
  Remote desktop logins are now persistent. If disconnected, your session stays preserved for seamless re-login.

- **Modern Dialogs:**
  System and application dialogs have been redesigned for improved usability, especially on narrow/mobile displays. File open/save dialogs now include advanced options like sorting, previews, and more.

---

## Additional Features

- **Files (Nautilus):**
  - Revamped places sidebar and network view.
  - File chooser improvements: larger thumbnails, choices menu, drag-and-drop, background menu support.
  - Performance boosts for file search and preview.
  - Extract bzip2 tar archives natively.

- **Wayland & Mutter Advances:**
  - Support for DRM lease protocol (improving VR handling).
  - Experimental color management.
  - Hardware encoding, fractional scaling enhancements, and more.
  - Option to build GNOME Shell/Mutter without X11, paving the way for a Wayland-only future.

- **Epiphany (GNOME Web):**
  - Navigation gestures toggle.
  - Improved dark mode (brightness/contrast).
  - New privacy report, bookmarks search, and autofill support.

![GNOME Web 47 Browser](https://upload.wikimedia.org/wikipedia/commons/thumb/1/1d/GNOME_Web_47_-_gnome.org.png/1024px-GNOME_Web_47_-_gnome.org.png)

- **General:**
  - Enhanced touchpad gesture support (three+ fingers).
  - Improved lock screen notifications, with centered time in 12h mode.
  - Improved monitor matching when reconnecting displays.
  - Workspace previews and window placement/focus improvements.
  - Enhanced support for legacy X11 cursor themes.
  - Tablet tool keybinding/action and pressure range support.
