# GNOME 47 “Denver” Release Notes

Project Bluefin includes GNOME 47, delivering a refined desktop experience with extensive customization, updated core applications, and improved system-wide aesthetics.

---

## 🎨 Customizable Accent Colors

GNOME 47 introduces native support for **accent colors**! You can now personalize your desktop with nine premium accent color choices:

* **Nine distinct shades** are available in *Settings* → *Appearance*.
* Accent colors apply seamlessly to highlights, buttons, sliders, and active states throughout GNOME shell and all modern GTK4 applications.
* Accessibility-aware color styling ensures high-contrast readability across all options.

---

## 📂 Nautilus & File Management Upgrades

The **Files** app (Nautilus) has received a massive round of feature updates:

### 1. Unified File Chooser
The file open/save dialogs have been redesigned from the ground up to share the exact styling and sorting controls of Nautilus. It now supports zooming, renaming, and rich search.

### 2. Tailored Sidebar
The Nautilus sidebar now consolidates system-wide locations (Home, Recent, Network, Trash) at the top. You can now easily hide default system directories like Music or Videos for a cleaner layout.

---

## ⚡ Performance & Hardware Acceleration

* **Screen Recording Hardware Encoding**: System screencasts now leverage Intel and AMD GPU hardware encoding directly, reducing CPU overhead dramatically.
* **Persistent Sessions**: Disconnected remote desktop connections remain active in the background, allowing you to reconnect and resume seamlessly.
* **Small Screen Usability**: UI elements scale cleanly on low-resolution or small displays, optimizing the layout of dialogs for mobile or compact screens.
